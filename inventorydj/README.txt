Inventory Management System JEDI
Universitas Trunojoyo Madura

Deskripsi:
Aplikasi web sederhana untuk mengelola data inventaris barang, kategori, dan supplier. Dibuat menggunakan PHP dan Tailwind CSS, serta dilengkapi fitur CRUD dan dark mode.

Fitur Utama:
- Login/logout
- CRUD Barang
- CRUD Kategori
- CRUD Supplier
- Dark mode (JavaScript)
- Tampilan mobile-friendly (Tailwind)
- Proteksi halaman dengan session

Akun Login:
Username: admin
Password: admin

Struktur Folder:
- login.php         -> Halaman login
- logout.php        -> Logout user
- index.php         -> Dashboard
- /barang/          -> CRUD Barang
- /kategori/        -> CRUD Kategori
- /supplier/        -> CRUD Supplier
- /config/          -> Koneksi & session
- README.txt        -> Panduan ini

Catatan Developer:
Proyek ini dibuat dengan pendekatan sederhana agar mudah dipahami oleh mahasiswa pemula dan bisa dijelaskan secara runtut saat demo. Tailwind digunakan via CDN dan JavaScript disisipkan langsung di halaman untuk memudahkan implementasi.
